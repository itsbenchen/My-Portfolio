Project: Facebook Clone

Built a Facebook clone using ReactJS and Firebase. Created various components of Facebook's UI, including the header, user's sidebar, main feed page, and widget section. Comes with full login capability (with Google login), where users can login to post on their feed. Users can also see what other users post in real-time, which is a prime feature of the original Facebook.

Project: RuneScape Bot Scripts

Designed and built robust scripts in Java using DreamBot's API and client for the botting community in Old School RuneScape. Scripts were designed to automate tasks and/or leveling, which allowed players to progress much faster. Worked with other community members in delivering a variety of scripts suited for their needs. Ensured scripts were well-designed and implemented, such that it would scale if needed and reduce chances of getting caught by an anti-bot catcher.

Project: Wumpus World

Made an AI using Python for the Wumpus World game that traversed the game environment (cave) with a goal of finding gold and exiting the cave successfully. Designed the AI in a way such that it adapted with its environment and chose the best decision depending on the knowledge it gathered to achieve the best possible performance. At the end of the project, the AI scored in the top 90th percentile in my course competition.

Project: Console-based Games

Implemented various games in C/C++, such as Conway's Game of Life, Blackjack, and Minesweeper. These games were lab projects done in my ICS 51 lab course, where I was given 2 hours to fully implement and demo the game. The purpose of these projects were to test our programming skills against the time, and to see if we can implement real-life applications/software.
